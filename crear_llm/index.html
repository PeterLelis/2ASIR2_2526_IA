01. Entorno de Trabajo Setup

Configuración inicial del entorno virtual y clonación del repositorio de herramientas base[cite: 4].

mkdir ejercicio-llm && cd ejercicio-llm
python3 -m venv venv && source venv/bin/activate
git clone https://github.com/ggerganov/llama.cpp.git
pip install -r llama.cpp/requirements.txt

02. Dataset Personalizado Dataset

Creación del archivo train.jsonl con las instrucciones de cocina para el entrenamiento del asistente[cite: 5].

nano data/train.jsonl

03. Entrenamiento del Modelo Fine-Tuning

Ejecución del entrenamiento LoRA configurado para 1000 iteraciones con una tasa de aprendizaje de 2e-5[cite: 6, 8].

python -m mlx_lm lora --model microsoft/Phi-3-mini-4k-instruct --train --data ./data --iters 1000 --batch-size 1 --learning-rate 2e-5 --adapter-path ./chef-bot-adapters

04. Fusión de Pesos Merge

Proceso de fusión (merge) de los adaptadores entrenados con el modelo base original para generar el modelo final[cite: 9].

mlx_lm.fuse --model microsoft/Phi-3-mini-4k-instruct --adapter-path ./chef-bot-adapters --save-path ./chef-bot-merged

05. Exportación a GGUF Quantize

Conversión del modelo fusionado al formato GGUF para optimizar su ejecución en diferentes plataformas[cite: 10, 11].

python llama.cpp/convert_hf_to_gguf.py ./chef-bot-merged --outfile chef-bot-f16.gguf --outtype f16

06. Evaluación Final Inference

Prueba de inferencia exitosa: El modelo responde correctamente sobre una receta de 'Ensalada Caprese' utilizando los ingredientes solicitados[cite: 12].
